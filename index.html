Concesionario

Tenemos una pagina principal con boton de inicio de sesion y o de registro.

Una vez se inice o registre el usuario en la pagina recogemos esos datos en una base de datos. Y se le redireje a una pagina donde se le muestra un catalogo de coches con un boton de comprar.

Tenemos los siguientes coches en stock: (que tambien estan alojados en una base de datos).

  • Seat Ibiza
  • Renault Clio
  • Ford Fiesta
  • Opel Corsa
  • Peugeot 208

El usario elige uno de los coches que se encuentra en el catalogo. Al elegir uno o varios de los siguientes modelos de coches se le redirige a una pagina de pago donde se le pide que introduzca sus datos bancarios y de envio. Y se actualiza el stock en la base de datos.

En el caso en el que al elegir uno de los coches seleccionados este sin stock ya que en el servidor hay un contador con el stock de cada uno de los modelos y se va actualizando conforme se van vendiendo los coches, se le informa al usuario de que el coche no esta disponible y se le redirige a la pagina principal.

Cliente

- Muestro la página con los botones de registro e inicio de sesión.
- El usuario puede navegar por el catálogo de coches.
- Si pulsa en el boton"Comprar", se le abre un formulario para introducir sus datos personales y bancarios.

Servidor

- Recibe los datos del registro o inicio de sesion del usuario.
- También gestiona la petición de compra:
- Comprueba en la base de datos si hay stock.
- Si hay stock, descuenta una unidad y confirma la compra.
- Si no hay stock, envío un mensaje de error y redirije al usuario a la página principal.

Base de datos

- Guardo los usuarios (nombre, email, contraseña…).
- Guardo los coches en stock (modelo, cantidad disponible, precio…).
- Guardo las compras realizadas (usuario, coche comprado, fecha, etc.).
- Actualizo los datos cada vez que alguien se registra, inicia sesión o compra un coche.
